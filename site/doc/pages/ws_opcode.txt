:sig
u8 ws_opcode()

:params
return value : opcode of the current WebSocket message

:desc
Returns the opcode of the message currently being handled by `WS(Request& context)`.

Common values are:

- `0x1` for text messages
- `0x2` for binary messages

:see
>websocket
